Flow Characteristics

  • Synchronous Flows: Payment processing, inventory checks (immediate response required)
  • Asynchronous Flows: Impact score calculation, batch processing (background tasks)
  • Real-time Flows: Leaderboard updates, notifications, inventory changes (WebSocket)
  • Event-driven Flows: Notifications triggered by events, webhooks for external systems
  • Batch Flows: Daily impact scoring, vendor reconciliation, analytics aggregation
